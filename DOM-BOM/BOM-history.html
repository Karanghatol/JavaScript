JavaScript Window History : 
    The window.history object contains the browsers history. 

    it can be written without and with window. object.

    Some methods:--------

    history.back() - same as clicking back in the browser
    history.forward() - same as clicking forward in the browser
    history.go() - his method is used to move the browser to a specific page in the session history. 
    
    history.pushState(state, title, url) - This method adds a new entry to the session history. 
    It takes three parameters: 
    state (a JavaScript object representing the state associated with the new history entry), 
    title (not widely used and often set to an empty string), and url (the new URL).

    history.replaceState(state, title, url):
    This method modifies the current entry in the session history without creating a new one. 
    It takes the same parameters as pushState.
